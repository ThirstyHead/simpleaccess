The <a> Element

As defined in the HTML Living Standard, the anchor element creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

Standard Link Example:
Visit the SimpleAccess Homepage to learn more.
<a href="https://simpleaccess.io">SimpleAccess Homepage</a>

WCAG Best Practices for Links

  • Descriptive Text: Avoid "Click Here" or "Read More". The link text alone should convey the destination.
  • Visual Distinctness: Links should be visually distinct from body text (usually via underlines and color contrast).
  • Focus Indicators: Always maintain a highly visible :focus state for keyboard users.

A11y Tip: External Links

If a link opens in a new window using target="_blank", it is a best practice to inform the user.
