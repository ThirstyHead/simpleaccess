🦈 Professional Windows Standard

Dominating the Windows Desktop.

JAWS (Job Access With Speech) is widely considered the most powerful screen reader for the Windows operating system. Developed by Freedom Scientific, it is the standard choice for professional environments. Mastering JAWS means understanding its robust "Virtual Cursor" and specialized HTML navigation commands. Start with these "Big Six" skills.

1. Launch and Close

JAWS is a dedicated application that runs alongside your Windows software. Use these standard shortcuts to control its lifecycle:

  • To Launch: Press Ctrl + Alt + J. You can also find it in the Start menu.
  • To Close: Press Insert + F4. A confirmation dialog will appear; press Enter to quit.

2. Mute (Silence)

If JAWS begins reading a long block of text (like a licensing agreement) and you've found what you need, use the standard silence command:

  • To Silence: Tap the Ctrl key. This stops speech immediately for the current element while keeping JAWS active for your next move.

3. Navigate by Headers

JAWS uses "Navigation Quick Keys" to skip between landmarks. This is the fastest way to satisfy WCAG SC 1.3.1.

  • Next Header H
  • Previous Header Shift + H
  • Header Level 1 through 6

4. Navigate by Links

Effective link navigation is key to meeting WCAG SC 2.4.4.

  • Next Link Tab
  • Previous Link Shift + Tab
  • List all Links Insert + F7

5. Navigate Form Fields

When you encounter a form, JAWS transitions from "Virtual Cursor" (reading) to "Forms Mode" (interaction).

  • Next Form Field F
  • Previous Form Field Shift + F
  • List all Form Fields Insert + F5

Note: JAWS will often play a "pop" sound when entering Forms Mode automatically.

SimpleAccess Insight: The JAWS Key

The JAWS Modifier Key is used for almost all advanced operations. Depending on your keyboard layout (Desktop vs Laptop), this is usually the Insert key or the Caps Lock key.

Testing Tip: Use Insert + Z to toggle the Virtual Cursor on or off. This allows you to see how your site behaves when the screen reader attempts to interact with it like an application rather than a document.
