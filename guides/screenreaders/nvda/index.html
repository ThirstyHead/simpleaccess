Universal Access for Windows.

NonVisual Desktop Access (NVDA) is a free, open-source screen reader for Windows that has become a favorite for both users and accessibility testers. Its adherence to standards and highly customizable "Browse Mode" make it a powerful tool for exploring the web. Master these "Big Six" skills to start your NVDA journey.

1. Launch and Close

NVDA is portable and can be installed or run from a USB drive. Once installed, use these commands to control the software:

  • To Launch: Press Ctrl + Alt + N at any time.
  • To Close: Press NVDA + Q. A dialog will appear asking if you want to exit or restart; press Enter to confirm.

2. Mute (Silence)

When NVDA is speaking and you need to pause or stop the speech immediately without turning the screen reader off:

  • Stop Speech: Tap the Ctrl key. This stops speech for the current item immediately.
  • Pause Speech: Tap the Shift key. Tapping it again will resume reading from the exact same spot.

3. Navigate by Headers

NVDA uses single-letter navigation in "Browse Mode" to help you scan the semantic structure of a page, satisfying WCAG SC 1.3.1.

  • Next Header H
  • Previous Header Shift + H
  • Jump to Header Level 1 through 6

4. Navigate by Links

Quickly jump between the interactive actions on a page to meet WCAG SC 2.4.4.

  • Next Link K
  • Previous Link Shift + K
  • Elements List (Links/Heads) NVDA + F7

5. Navigate Form Fields

NVDA automatically switches between "Browse Mode" (reading) and "Focus Mode" (typing) when you enter a form field.

  • Next Form Field F
  • Previous Form Field Shift + F
  • Force Focus Mode NVDA + Space

SimpleAccess Insight: The "NVDA" Key

The NVDA Modifier Key is essential for almost all commands. By default, it is the Insert key (found above the arrow keys or on the numpad) or the Caps Lock key.

Testing Tip: Use the Elements List (NVDA + F7) on this page. It's an excellent way to verify that your heading hierarchy and link descriptions are clear and meaningful for non-visual users.
